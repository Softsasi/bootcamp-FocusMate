## ğŸ“¦ Installation

### Prerequisites

- **Node.js** (v18 or higher)
- **npm** or **yarn**

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Softsasi/bootcamp-FocusMate.git
   cd bootcamp-FocusMate
   ```

2. **Install dependencies**
